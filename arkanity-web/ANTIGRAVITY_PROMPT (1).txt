**TASK: Fix Netlify Forms - They're currently broken and showing error modal**

**THE PROBLEM:**
The forms on arkanity.agency are failing with error: "There was a problem sending. Please try again or email us at info@arkanity.agency"

Root cause: The `js/main.js` file is intercepting form submissions via JavaScript, which conflicts with Netlify Forms' native HTML submission.

**THE FIX NEEDED:**

Please make these 3 changes to the website:

## 1. DISABLE THE MAIN.JS SCRIPT

In `index.html`, find this line (near the end, around line 1632):
```html
<script src="js/main.js?v=3.0" defer></script>
```

Replace it with:
```html
<!-- <script src="js/main.js?v=3.0" defer></script> -->
<!-- DISABLED: Conflicts with Netlify Forms native submission -->
```

## 2. ADD SUCCESS REDIRECTS TO BOTH FORMS

**Form 1 - Lead Magnet Form (around line 961):**

Find:
```html
<form class="space-y-4" id="lead-magnet-form" data-netlify="true" name="lead_magnet" method="POST">
```

Replace with:
```html
<form class="space-y-4" id="lead-magnet-form" data-netlify="true" name="lead_magnet" method="POST" action="/success.html">
```

**Form 2 - Consultation Form (around line 1290):**

Find:
```html
<form class="space-y-6" id="contact-form" data-netlify="true" name="consultation" method="POST">
```

Replace with:
```html
<form class="space-y-6" id="contact-form" data-netlify="true" name="consultation" method="POST" action="/success.html">
```
